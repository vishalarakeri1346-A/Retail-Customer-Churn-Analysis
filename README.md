# Retail Customer Churn Analysis

## Project Overview
Retail Customer Churn Analysis — end-to-end project demonstrating data cleaning, feature engineering, machine learning, SQL KPIs, and model inference.  
Goal: identify drivers of churn and build a model to predict which customers will churn.

## Repository Structure
See `README` for full structure. Key folders:
- `data/raw/` — place `customer_churn.csv` here.
- `data/processed/` — generated datasets.
- `scripts/` — all runnable Python scripts.
- `models/` — saved best model.
- `sql/` — SQL queries for KPIs.
